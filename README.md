# 📈 Stock Portfolio Tracker 

## 📌 Project Overview

This project is a simple **Stock Portfolio Tracker** developed using **Python**. The program allows users to enter stock names and quantities, calculates the total investment value based on predefined stock prices, and displays the overall portfolio value.

The stock prices are stored in a hardcoded dictionary, making the project simple and beginner-friendly for understanding Python concepts such as dictionaries, arithmetic operations, user input, and file handling.

---

## ✨ Features

* User inputs stock names and quantities
* Uses a predefined dictionary for stock prices
* Calculates total investment value
* Displays portfolio summary and total amount
* Optional feature to save results in a `.txt` or `.csv` file
* Simple console-based interface (Terminal / Python Compiler)

## ▶️ How to Run

1. Open any Python compiler (**online or offline**).
2. Copy and paste the Python code into the compiler.
3. Click **Run / Execute**.
4. Enter stock names and quantities when prompted.
5. View the total investment value of your portfolio.

---

## 🎯 Example Workflow

* The user enters a stock name (e.g., **AAPL**, **TSLA**).
* The user enters the quantity of shares.
* The program checks stock prices from a predefined dictionary.
* The total investment amount is calculated and displayed.
* Optionally, results can be saved to a text or CSV file.
